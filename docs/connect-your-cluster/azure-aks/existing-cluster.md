# Azure AKS — I Already Have a Cluster

## What you need before you start

- An Azure account with an active AKS cluster
- Your Azure account must have at minimum the **Azure Kubernetes Service Cluster User Role** on the cluster

## Connect the cluster

1. In CaseDesk, go to **Clusters → Connect cluster → Azure AKS**
2. Click **Sign in with Microsoft** — you'll be redirected to Microsoft OAuth
3. Authorise CaseDesk to list your AKS clusters
4. Select the cluster to connect
5. Click **Connect cluster →**

Your OAuth tokens are encrypted and stored securely. CaseDesk refreshes them automatically.
