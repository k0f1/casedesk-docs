# What is CaseDesk

CaseDesk lets you deploy open-source AI models directly into your own cloud infrastructure — AWS EKS or Azure AKS — and use them via an OpenAI-compatible API endpoint.

You bring the cluster. CaseDesk handles the deployment, scaling, and endpoint routing.
